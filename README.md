# Vue Dashboard System

A simple dashboard system built with **Vue.js**, featuring a permission-based layout system.  
Backend support is provided by **Next.js**, showcasing both frontend and backend integration.

---

## 🌐 Live Demo
[View Project](https://vue-demo-dashboard.vercel.app/)

---

## 📦 Dependencies
- [Vue Router](https://router.vuejs.org/) – for navigation & routing  
- [Vuex](https://vuex.vuejs.org/) – for state management  
- [Shadcn Vue](https://shadcn-vue.com/) – for modern UI components
